Basic template

How to start a page with G.R.I.D ?

Use the HTML below to begin working with a minimal G.R.I.D document.

<!DOCTYPE html>
<html>
    <head>
        <meta http-equiv="content-type" content="text/html; charset=utf-8" />
        <meta name="viewport" content="width=device-width; initial-scale=1.0" />
        <!-- The above meta tags *must* come first in the head; any other head content must come *after* these tags -->
 
        <title>G.R.I.D minimal template</title>
 
        <!-- G.R.I.D CSS -->
        <link rel="stylesheet" type="text/css" href="css/g.r.i.d.min.css" />
        
        <!-- You can add here some Javascript tweaks like respond.js and html5shiv.js for IE8 support of HTML5 and media queries -->
        <!--[if lt IE 9]>
           <script src="https://oss.maxcdn.com/html5shiv/3.7.2/html5shiv.min.js"></script>
           <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
       <![endif]-->
       
    </head>
   
    <body>
        <!-- This div is very important ! All your G.R.I.D documents must have a div with an id "page" who englobe all your body content -->
        <div id="page">
            <!-- You can use it for your header -->
            <header id="header-wrapper">
                <!-- This div is also important. Before insert any content you have to create this div -->
                <div class="container">
                    <h1>Welcome !</h1>
                </div>
            </header>
            
            <!-- You can use it for your contents -->
            <div id="main-wrapper">
                <div class="container">
                </div>
            </div>

            <!-- You can use it for your footer -->
            <footer id="footer-wrapper">
                <div class="container">
                </div>
            </footer>
        </div>
        
        <!-- jQuery (necessary for G.R.I.D's JavaScript plugins) -->
        <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
        <!-- Include all plugins -->
        <script type="text/javascript" src="js/g.r.i.d.min.js"></script>
    </body>
</html>

This template structure is highly recommended to start a G.R.I.D document ! All example templates use this structure.

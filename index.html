Welcome to G.R.I.D

The simple way to Get Responsive elements In your Design ! G.R.I.D will help you to build a responsive template for your website, or make your existing template responsive ! G.R.I.D has also a Javascript version, which comes with a lot of jQuery plugins to help you to boost up the interactivity of your website !

Easy to use

G.R.I.D is the easy way to make your website's template responsive ! You just have to create a HTML element with classes or IDs like grid, menu or panel and let the CSS and the JS do the rest !

Designed for all

G.R.I.D is made for all types of designs, folks of all skill levels and projects of all sizes. G.R.I.D CSS and G.R.I.D JS (jQuery plugins) allows you to easily and fastly create responsive elements in your template !

It's just FREE !

G.R.I.D is a free Open Source project under the MIT License. It's hosted, developed, and maintained on GitHub. You can see this project on GitHub for opening new issues or submitting a pull request !

CSS Preprocessor

G.R.I.D is now powered by Less ! The one of the most popular CSS preprocessors. You can quickly get started with precompiled .css files, or use .less source files to edit and compile your own version of G.R.I.D to create something more unique !

Ready for all devices

G.R.I.D adapts your websites and applications for various screen sizes, from phones to tablets, desktops and TVs, by using CSS media queries.

Javascript Plugins and CSS Helpers

G.R.I.D comes with a lot of jQuery plugins (Dropdowns, Progress bars, Tooltips, Sliders, etc...) and some CSS helpers likes the FontAwesome icons pack and the awesome CSS animations pack !
